# funds-management-app-demo
funds-management-app-demo that automates fund requests, approvals, and disbursements, eliminating manual paperwork and improving operational efficiency. 

## Problem
Manual fund requests and approvals were handled via paper and informal communication,
leading to delays, errors, and lack of traceability.

## Solution
A role-based mobile application where:
- Staff request funds from their dashboard
- MD approves or declines requests
- Fund disburser only sees approved requests

## Features
- Fund request tracking (Pending / Approved / Declined)
- Role-based dashboards
- Real-time updates
- Secure approval workflow

## Tech Stack
- Flutter & Dart
- Supabase (Auth, Database)
- RESTful APIs

## Architecture Overview
Flutter UI → Supabase → Backend APIs

This repository is a **demo showcase**. The full production source code is private
